introduction to git

git is an open source version control system that is used to track changes made to our code

advantages of using git

  • fast and efficient
  • distributed version control
  • enhanced collaboration
  • compatibility
  • data integrity

basic git command

  • git init : used to initialize a git repository
  • git add . : used to stage changes made to our code
  • git status used to check the status of our code
  • git commit -m "commit message" :this is used to save the staged changes with a descriptive message
  • git remote add origin this is used to connect our local repo to our remote repo on github
  • git push origin main this is used to push your code from local repo to remote repo on github
